# Backend CréditoSquid

## Levantar Ambiente Virtual
1. Abrir PowerShell en esta carpeta.
2. Ejecutar: `.\venv\Scripts\Activate.ps1`

## Ejecutar el Proyecto
- Comando: `python wsgi.py`
- URL local: `http://127.0.0.1:5000`

## Dependencias
- Si agregas una librería: `pip freeze > requirements.txt`
- Para instalar todo: `pip install -r requirements.txt`
